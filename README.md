# Personal Portfolio Website

This is the **personal portfolio website of Syahrill Norizan**, designed to showcase skills, projects, and experience. It includes a dynamic project section, responsive design, and interactive features such as an image slider for project previews.

---

## Demo

You can view the live here:  
https://syahrillnzr.github.io/

---

## Overview

This website is a personal portfolio designed to present:

- Professional background and skills  
- Completed projects with details and screenshots  
- Contact information for collaboration or inquiries  

The website uses dynamic rendering for projects, an image slider for visual presentation, and a responsive layout for mobile and desktop.

---

## Features

- Responsive and mobile-friendly design  
- Dynamic project section with image sliders  
- Highlights for skills, technologies, and project goals  
- Interactive navigation and smooth scrolling  
- Contact section for inquiries  

---

## Technologies Used

- HTML5  
- CSS3
